🚀 5. STRICT DATA FLOW (The Python Pipeline Loop)



Der Datenfluss für die neuen Module folgt einem klaren Client-Microservice-Pattern:



    Operator Input: User gibt im Frontend Daten für Synergy oder Chronos ein.



    Data Serialization: `python\_client.js` packt den Request und sendet ihn authentifiziert an den FastAPI Endpoint.



    The Deep Calculation: Der Python-Core (Serverless/Dockerized) berechnet Matrizen, vergleicht Vektoren und ermittelt Reibungsverluste oder temporale Peaks.



    JSON Response: Python retourniert ein strukturiertes, validiertes JSON-Datenpaket.



    Visual Rendering: Das Frontend übersetzt die Daten verzögerungsfrei in Gunmetal-Cards, Deep-Purple-Akzente und OLED-Black Dashboards.
