# Synergy Module

## Overview

The Synergy Module displays numerological compatibility analysis between two persons. It shows life path numbers, master numbers, and combined energy readings in an elegant glassmorphism card design.

**Now Interactive:** Enter two birthdates, click "Synergie prüfen", and see live calculations directly in the browser!

## Dashboard Integration

The Synergy Matrix Card is integrated into the main dashboard (`/dashboard/index.html`) and renders dynamically via JavaScript.

### Data Flow

1. **Source**: `/docs/S3_Data/synergy_input_raw.json` (generated by DATA-MAKER)
2. **Renderer**: `/dashboard/render_dashboard.js` - `renderSynergyMatrix()` function
3. **Styling**: `/shared/ui/components.css` - Synergy-specific CSS classes

### Features

- **Interactive Input**: Two input sections for Person A and Person B
- **Live Calculation**: Calculates life path numbers in real-time via browser
- **Smart Date Input**: Auto-formats dates to TT.MM.JJJJ while typing
- **Glassmorphism Design**: backdrop-filter blur, subtle purple glow effects
- **Dynamic Data Loading**: Fetches initial JSON data from docs folder
- **XSS-Safe Rendering**: Uses `dom_utils.js` (Law 3 compliant - no innerHTML)
- **Skeleton Loading**: Shows loading state while data is fetched
- **Responsive Grid**: Person cards adapt to screen size

### Displayed Information

- **Synergy Title**: "Kreativ & Kommunikativ" (from JSON)
- **Combined Energy**: Numerical value (e.g., 8)
- **Person Cards**:
  - Name and birthdate
  - Life path number (large display)
  - Master number badge (if applicable)
